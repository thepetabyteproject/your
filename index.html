    your

    your stands for Your Unified Reader. This library reads sigproc filterbanks, psrfits, and psrdada formats and can go from one format to another.

    Format Read Write
    filterbank
    psrfits
    psrdada

    The inspiration for the name comes from the Daily Dose of Internet videos.

    Installation

    First you need to install psrdada-python. Once psrdada is installed, git clone the repo and use setup.py to install your.

    git clone https://github.com/devanshkv/your.git
cd your
python setup.py install
